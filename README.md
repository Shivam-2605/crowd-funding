# 💰 Crowd Funding Platform

## 📌 Overview
This project is a **Crowdfunding Platform** that allows users to create campaigns and raise funds for various causes.  
It provides a seamless and transparent way for individuals to **support projects, donate securely, and track contributions**.

## 🚀 Features
- 🧑‍💻 User-friendly interface to explore campaigns  
- 💸 Secure donation system  
- 📊 Transparent tracking of funds  
- 📢 Campaign sharing functionality  
- 🛠️ Admin controls for managing campaigns  
- 📱 Responsive UI  

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Flask / Node.js *(update if needed)*  
- **Database:** MongoDB *(or your DB)*  
- **Version Control:** Git & GitHub

## ⚙️ Installation & Setup
### 1️⃣ Clone the Repository
```bash
git clone https://github.com/Shivam-2605/crowd-funding.git
cd crowd-funding
```

### 2️⃣ Install Dependencies
```bash
pip install -r requirements.txt
```
*(or use `npm install` if Node.js)*

### 3️⃣ Run the Project
```bash
python app.py
```

## 🌐 Deployment
### Frontend (GitHub Pages)
1. Move `index.html` to root or select `/frontend` in settings
2. Go to **Settings → Pages**
3. Enable deployment from `main` branch
